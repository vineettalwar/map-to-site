# Design Notes

## Admin UI

- Gemini-inspired single-field input on `/`
- Full-viewport hero with soft radial gradients
- Large rounded input shell with glow on focus
- Minimal chrome: product label only

## Landing Template

- Dark hero with optional background photo
- Light content sections with card-based highlights
- Review wall on dark background
- Contact card with icon rows

## Tokens

Uses Shadcn neutral theme via CSS variables in `app/globals.css`.
